================================================================================

TEST THESE FEATURES:

[1] Navigation
    - Click "Nail Lengths" in the header
    - Page should smoothly scroll to that section

[2] Select Nail Length
    - Click any nail length card (e.g., "Medium")
    - Card should get a pink border
    - See "Length selected!" message

[3] Select Design Tier
    - Click "Select Tier 3" button
    - Card should highlight with pink border
    - See "Tier 3 selected!" message

[4] Gallery Lightbox
    - Scroll to "Pre-Made Designs"
    - Click any design image
    - Full-screen modal opens
    - Try these:
      * Click Next/Previous buttons
      * Press arrow keys on keyboard
      * Press ESC to close

[5] Mobile Menu
    - Resize browser to mobile size (drag right edge inward)
    - Click hamburger icon (three lines, top-right)
    - Menu should slide in from left
    - Click outside to close

[6] Book Appointment
    - Click "Book In-Person Appointment" button
    - Modal opens with form
    - Fill out all fields
    - Submit form
    - Success message appears!

[7] Shipped Sets
    - Click "Order Shipped Sets" button
    - Fill out shipping form
    - Submit and see confirmation

================================================================================
  GETTING HELP
================================================================================

If something doesn't work:

1. Check browser console (F12) for errors
2. Read "JAVASCRIPT_GUIDE.md" troubleshooting section
3. Make sure all files are in same folder
4. Try a different browser
5. Clear browser cache

Common issues:
  - "Script not loading" → Check if script.js is in same folder
  - "Forms not working" → Fill ALL required fields (marked with *)
  - "Mobile menu stuck" → Press ESC or refresh page
  - "Nothing is interactive" → Make sure script.js loaded (check F12 console)
